# Hedy App Twin

Static HTML twin of the Hedy macOS app, deployed to https://apptwin-six.vercel.app
(Vercel auto-deploys `main` of this repo).

- `hedy-{screen}-{light|dark}-1440.html` — one self-contained page per screen/state,
  fixed 1440×900 (Figma-capturable via html.to.design), medium breakpoint <1280px.
- `app.html` — interactive single-page prototype (all screens, client-side nav,
  dark-mode toggle, menus/modals). `index.html` — hub linking everything.
- `generator/` — source of truth. Pages are generated by `generator/build_twins.py`
  from the `hedy_mobile` codebase (branch `app-twin-2026-07-08`); never hand-edit
  the HTML. `generator/JOURNAL.md` documents the sync-with-main procedure and log.

1:1 rule: visuals mirror the hedy_mobile code exactly; only mock data is invented.
